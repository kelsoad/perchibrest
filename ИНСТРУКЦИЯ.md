# Как добавить товар в каталог PerchiBrest

Инструкция без скриптов: фото готовятся в браузере, товар добавляется в файл `data/catalog.json`.

---

## Что нужно для одного товара

В папку `images/catalog/` кладутся **3 файла** с одинаковым именем (без расширения):

| Файл | Назначение | Размер |
|------|------------|--------|
| `название.jpg` | Полное фото (открывается по клику) | до 1200 px по длинной стороне, JPEG |
| `название-card.jpg` | Миниатюра в каталоге | **224×224 px**, JPEG |
| `название-card.webp` | Миниатюра (лёгкая версия) | **224×224 px**, WebP |

**Пример** для товара «Перчатки синие»:

```
images/catalog/perchatki-sinye.jpg
images/catalog/perchatki-sinye-card.jpg
images/catalog/perchatki-sinye-card.webp
```

Имя файла — латиница или цифры, без кириллицы. Пробелы лучше не использовать (или как в существующих файлах: `101 600.jpg`).

---

## Шаг 1. Подготовить большое фото

1. Сфотографируйте или возьмите готовое фото товара на **светлом или нейтральном фоне**.
2. Важное — **по центру** (края могут обрезаться в миниатюре).
3. Сохраните как JPEG, например: `perchatki-sinye.jpg`.

### Сжать большое фото через интернет (по желанию)

Сайт: **https://squoosh.app**

1. Откройте Squoosh в браузере.
2. Перетащите файл `perchatki-sinye.jpg`.
3. Справа выберите формат **MozJPEG**, качество **80–85**.
4. Если фото очень большое — уменьшите ширину до **800–1200 px** (Resize).
5. Скачайте файл и положите в `images/catalog/perchatki-sinye.jpg`.

---

## Шаг 2. Сделать миниатюру 224×224 (JPEG)

Сайт: **https://squoosh.app** (или **https://www.iloveimg.com/resize-image**)

### Вариант A — Squoosh (рекомендуется)

1. Откройте **https://squoosh.app**.
2. Загрузите `perchatki-sinye.jpg`.
3. Включите **Resize**:
   - ширина: **224**
   - высота: **224**
   - метод обрезки: **cover** / заполнить кадр (чтобы был квадрат).
4. Формат справа: **MozJPEG**, качество **80**.
5. Скачайте как: `perchatki-sinye-card.jpg`.
6. Положите в `images/catalog/`.

### Вариант B — iLoveIMG

1. Откройте **https://www.iloveimg.com/resize-image**.
2. Загрузите фото → размер **224×224** → обрезка по центру.
3. Скачайте и переименуйте в `perchatki-sinye-card.jpg`.

---

## Шаг 3. Сделать миниатюру WebP

Сайт: **https://squoosh.app** или **https://cloudconvert.com/jpg-to-webp**

### Squoosh

1. Снова откройте **исходное** фото (или уже готовый `perchatki-sinye-card.jpg`).
2. Resize: **224×224** (как в шаге 2).
3. Формат справа: **WebP**, качество **80**.
4. Скачайте как: `perchatki-sinye-card.webp`.
5. Положите в `images/catalog/`.

### CloudConvert

1. **https://cloudconvert.com/jpg-to-webp**
2. Загрузите `perchatki-sinye-card.jpg`.
3. Скачайте WebP и переименуйте в `perchatki-sinye-card.webp`.

---

## Шаг 4. Добавить товар в `data/catalog.json`

1. Откройте файл `data/catalog.json` в блокноте или редакторе.
2. Найдите массив `"products": [ ... ]`.
3. В конец списка (перед закрывающей `]`) добавьте запятую после предыдущего товара и вставьте новый блок:

```json
{
  "id": "hb-perchatki-sinye",
  "category": "hb",
  "title": "Перчатки синие",
  "priceLabel": "от 0,50 руб.",
  "image": "images/catalog/perchatki-sinye.jpg",
  "description": "Краткое описание для всплывающего окна.",
  "pricing": "до 100 пар - 0,60 руб.; свыше 1000 пар - 0,50 руб. ВСЕ ЦЕНЫ УКАЗАНЫ БЕЗ НДС"
}
```

4. Вверху файла обновите дату: `"updated": "2026-07-08"` (сегодняшняя дата).

### Поля товара

| Поле | Что писать |
|------|------------|
| `id` | Уникальный код латиницей, например `hb-noviy-tovar` |
| `category` | Категория фильтра (см. таблицу ниже) |
| `title` | Название на карточке |
| `priceLabel` | Цена под названием, например `от 0,36 руб.` |
| `image` | Путь к **большому** фото: `images/catalog/имя.jpg` |
| `description` | Описание во всплывающем окне |
| `pricing` | Полная сетка цен (тот же текст, что в прайсе) |

### Категории (`category`)

| Значение | Фильтр на сайте |
|----------|-----------------|
| `hb` | ХБ |
| `obliv` | Облив |
| `Latex` | Резиновые |
| `kragi` | Краги+рукавицы |
| `vetos` | Ветошь, мыло, паста |

---

## Шаг 5. Загрузить на хостинг

Через FTP или файловый менеджер хостинга загрузите:

1. `images/catalog/perchatki-sinye.jpg`
2. `images/catalog/perchatki-sinye-card.jpg`
3. `images/catalog/perchatki-sinye-card.webp`
4. `data/catalog.json` (обновлённый)

---

## Шаг 6. Проверить

1. Откройте сайт в браузере.
2. Раздел **КАТАЛОГ** — новый товар в сетке, фото **одного размера** с остальными.
3. Клик по фото — открывается **большое** изображение.
4. Фильтр по категории — товар в нужной группе.

Если миниатюра не видна — проверьте имена файлов: должны быть `-card.jpg` и `-card.webp` с тем же именем, что у основного `.jpg`.

---

## Краткая памятка

```
1. Большое фото     → images/catalog/имя.jpg
2. Squoosh 224×224  → images/catalog/имя-card.jpg   (JPEG)
3. Squoosh 224×224  → images/catalog/имя-card.webp  (WebP)
4. Запись в         → data/catalog.json
5. Загрузка на      → хостинг (все 4 файла)
```
